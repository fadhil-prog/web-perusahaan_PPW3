# web-perusahaan_PPW3
Fadhil Akbar (2315061037)

web ini adalah web yang bertujuan untuk memberikan informasi ke masyarakat terkait sebuah perusahaan. HTML tersebut dibuat dengan framework tailwind yang sanagat fleksibel untuk digunakan karena dapat menerapkan gaya langsung di HTML tanpa beralih file, fleksibilitas desain yang tinggi berkat pendekatan utility-first, serta desain responsive yang mudah dengan sistem grid bawaan
